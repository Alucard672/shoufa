# 云函数上传方法

## 方法一：通过云开发控制台上传（推荐）

1. **打开云开发控制台**
   - 在微信开发者工具顶部菜单栏，点击"云开发"按钮
   - 或者点击工具栏的"云开发"图标

2. **进入云函数管理**
   - 在云开发控制台中，点击左侧菜单的"云函数"

3. **创建/上传云函数**
   - 点击"新建云函数"按钮
   - 输入函数名称：`createIssueOrder`
   - 点击"确定"创建
   - 创建后，点击函数名称进入详情页
   - 点击"上传代码"按钮
   - 选择本地文件夹：`cloudfunctions/createIssueOrder`
   - 点击"确定"上传

4. **安装依赖**
   - 上传完成后，点击"安装依赖"按钮
   - 等待安装完成

5. **重复步骤3-4**，上传其他云函数：
   - `createReturnOrder`
   - `initDatabase`（可选）

## 方法二：通过文件管理器上传

1. **打开云开发控制台** → **云函数**

2. **创建云函数**
   - 点击"新建云函数"
   - 输入函数名称（如：`createIssueOrder`）
   - 点击"确定"

3. **上传代码**
   - 在函数列表中，点击函数名称
   - 在代码编辑器中，点击"上传代码"
   - 选择本地文件夹路径

## 方法三：直接在控制台创建（简单但需要手动输入代码）

1. **打开云开发控制台** → **云函数**

2. **新建云函数**
   - 点击"新建云函数"
   - 输入函数名称：`createIssueOrder`
   - 点击"确定"

3. **复制代码**
   - 打开本地文件 `cloudfunctions/createIssueOrder/index.js`
   - 复制全部代码
   - 粘贴到云函数编辑器中
   - 保存并部署

4. **安装依赖**
   - 点击"安装依赖"按钮

## 快速操作步骤（推荐方法一）

### 上传 createIssueOrder

1. 云开发控制台 → 云函数 → 新建云函数 → 输入 `createIssueOrder`
2. 创建后点击函数名 → 上传代码 → 选择 `cloudfunctions/createIssueOrder` 文件夹
3. 上传完成后 → 安装依赖

### 上传 createReturnOrder

1. 云开发控制台 → 云函数 → 新建云函数 → 输入 `createReturnOrder`
2. 创建后点击函数名 → 上传代码 → 选择 `cloudfunctions/createReturnOrder` 文件夹
3. 上传完成后 → 安装依赖

## 注意事项

- 云函数名称必须与代码中调用的名称完全一致
- 上传后需要安装依赖（npm install）
- 如果修改了代码，需要重新上传
- 可以在云函数列表中查看上传状态和日志

## 验证上传成功

上传完成后，在云函数列表中可以看到：
- ✅ 函数名称
- ✅ 运行环境
- ✅ 最后更新时间

点击函数名称可以：
- 查看代码
- 查看日志
- 测试调用
- 查看监控数据
